. Boutique en ligne – Projet Semestre 3

Ce projet est un petit site web de boutique en ligne réalisé dans le cadre de mon troisième semestre universitaire.

Le site permet de vendre des vêtements et des montres pour homme à travers une interface simple.

. Fonctionnalités

Sur ce site, l'utilisateur peut :

Consulter les produits disponibles dans un catalogue

Naviguer facilement entre les différentes pages grâce à un menu de navigation

Voir les détails des produits

Ajouter des produits au panier

Consulter le panier

Le catalogue et le menu permettent de faciliter la navigation et le déplacement sur le site.

 . Objectif

  L'objectif de ce projet est de créer une boutique en ligne simple et de mettre en pratique les connaissances en développement web.
